# Boss Hiring Telegram Bot SOP

## 1. 首次关注

1. 用户进入 Bot。
2. Bot 展示 Boss Hiring 公司介绍。
3. 用户选择语言：中文 / English / ភាសាខ្មែរ。
4. Bot 展示主菜单。

## 2. 求职者 SOP

1. 点击「我要找工作」。
2. 选择提交方式：
   - 在线填写资料
   - 下载简历模板后上传
   - 已有简历，直接上传
3. 用户提交资料或上传文件。
4. 文件上传作为 Airtable 附件。
5. AI 提取简历内容，生成摘要、标签、缺失项。
6. 结构化数据进入 Airtable Candidates 表。
7. 内部 Telegram 群收到新求职者提醒。
8. HR 人工审核并更新状态。

## 3. 企业招聘 SOP

1. 点击「企业我要招聘」。
2. 选择提交方式：
   - 在线填写招聘需求
   - 下载招聘需求模板后上传
   - 已有 JD，直接上传
3. 企业提交资料或上传文件。
4. 文件上传作为 Airtable 附件。
5. AI 提取岗位信息，生成岗位摘要、候选人标签、缺失项。
6. 结构化数据进入 Airtable Jobs 表。
7. 内部 Telegram 群收到新企业招聘提醒。
8. 顾问人工确认合作规则后推进推荐。

## 4. Boss 来了 SOP

1. 点击「了解 Boss 来了」。
2. 查看栏目介绍、往期访谈、申请企业采访或品牌合作。
3. 用户提交合作意向。
4. 数据进入客服跟进流程（或记入 Airtable）。
5. 内部 Telegram 群提醒负责人跟进。

## 5. 数据表

Airtable 建立两张核心表：

- Candidates (求职者简历表)
- Jobs (企业岗位需求表)

每条记录都保留：

- 提交时间
- Telegram ID
- Telegram 用户名
- 语言
- 来源入口
- 文件链接
- AI 摘要
- AI 标签
- 缺失项
- 当前状态
- 跟进负责人
- 备注
